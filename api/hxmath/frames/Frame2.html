package hxmath.frames

from IFrame2

A 2D affine frame with an offset and an angle. Frames allow you to transform between world/outer coordinates and local/inner coordinates in a form that is more meaningful/less verbose compared to a matrix. Frames can be concatenated, inverted, and converted to matrices.
